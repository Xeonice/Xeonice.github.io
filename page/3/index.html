后代选择器

可用来寻找特定元素或元素组的后代。后代选择器由其他两个选择器中间的空格表示。在此事例中，css 属性仅影响 blockquote 的后代的 <p> 元素。


要想寻找更特定的元素，可以使用 ID选择器 和 类选择器 。顾名思义，这两种选择器用于寻找那些具有指定 ID 或 类名 的元素。下方示例，第一条选择 ID = intro 的元素，第二条选择 class = data-posted 的元素。
